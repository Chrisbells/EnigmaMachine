This is an emulator of the enigma 1 machine which was used by Germany in ww2. The machine works by passing an electrical signal through rotors that are wired to swap around the letters. When a letter is typed the first rotor spins, eventually when the first rotor fully spins the next one over turns and so on.

To use this set the rotors and their positions and press update settings or use the defaults. You can also set up the plugboard by creating pairs by pressing the buttons at the bottom clicking the bottom keys will input a letter and you'll see the matching letter light up

The main issue with the enigma machine was the fact that a character input could never have itself as the output this means that unlike modern cryptography it isn't completely random. This meant that people trying to crack the code could guess a phrase that will come up in the message and then check for where the letters don't match giving a good starting point to crack it. This vulnerability and approach lead to a machine called the Bombe which automatically guessed and checked through enigma configuration combinations.

Another issue with the enigma machine was how the keys or the machine configuration was key distribution. If a spy got their hands on the keys and shared them there was nothing Germany could do if they Were unaware. Modern cryptography and encryption rely on public and private keys with a public key you can encrypt but to decrypt the private key is required. This allows 2 parties talking to each other to create a private-public key pair and share their public keys and not have to worry if anyone gets their hands on the public key. The only issue is that doing so on mechanical hardware would've been extremely difficult
